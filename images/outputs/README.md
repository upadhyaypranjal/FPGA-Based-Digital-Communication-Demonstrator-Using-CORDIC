# GUI Output Gallery

This directory contains representative screenshots captured from the FPGA CORDIC Digital Communication Demonstrator during hardware execution.

The images demonstrate the graphical interface, waveform visualization, constellation diagrams, and CORDIC verification results generated using the FPGA implementation running on the Vicharak Shrike Lite platform.

# GUI Outputs

The `gui_outputs` directory contains screenshots captured directly from the Python Host GUI.

---

## 1. CORDIC Calculator

Location

```
gui_outputs/
└── cordic_calculator/
```

This folder demonstrates the standalone CORDIC verification utility included within the GUI.

### Degree Mode

Shows FPGA-generated SIN, COS and TAN values for angles entered in degrees.

Images include

- Positive angle verification
- Large angle normalization
- Error comparison with Python math library

Examples

- 60°
- 1020°

---

### Radian Mode

Shows hardware computation using radian inputs.

Examples

- π/4
- π/2

Each screenshot compares

- FPGA CORDIC output
- Expected mathematical value
- Absolute error

## 2. BPSK Demonstration

Location

```
gui_outputs/
└── bpsk/
```

Demonstrates Binary Phase Shift Keying using FPGA-generated carrier phase values.

Three input modes are illustrated.

---

### Binary Input

User manually enters a binary sequence.

Screenshots include

- Complete GUI
- Waveform visualization
- I/Q constellation

Shows

- Baseband bits
- Carrier waveform
- Modulated RF waveform
- Phase transitions between 0° and 180°

---

### Random Input

Random binary data generated by the GUI and transmitted through the FPGA.

Screenshots demonstrate

- Random sequence generation
- Corresponding waveform
- Dynamic constellation transitions

---

### Text Input

ASCII text converted into binary before FPGA transmission.

Example

```
COMPUTER
```

Illustrates

- Text-to-binary conversion
- BPSK modulation
- Waveform generation
- Constellation updates

---

## 3. QPSK Demonstration

Location

```
gui_outputs/
└── qpsk/
```

Demonstrates Quadrature Phase Shift Keying using FPGA-generated CORDIC outputs.

The GUI groups input bits into symbol pairs and maps them to four constellation points.

Supported phase states

- 45°
- 135°
- 225°
- 315°

Each operating mode includes

- Complete GUI
- Waveform visualization
- Constellation animation

---

### Binary Input

Manual binary sequence.

Demonstrates

- Symbol mapping
- Carrier phase selection
- Four-state constellation

---

### Random Input

Random bitstream generated by the GUI.

Shows

- Random QPSK symbols
- Real-time waveform evolution
- Dynamic constellation transitions

---

### Text Input

ASCII text converted into binary and grouped into dibits.

Illustrates

- Character encoding
- Symbol generation
- Phase mapping
- Complete QPSK modulation

# Thonny Shell Outputs

Location

```
thonny_shell_outputs/
```

This folder contains console outputs captured from the RP2040 firmware while interacting directly with the FPGA through the Thonny IDE.

These screenshots verify the complete hardware communication pipeline without using the graphical interface.

---

## Degree Mode

Image

```
output1
```

Demonstrates FPGA CORDIC execution using degree-based angle inputs.

Example test cases include

- 30°
- 215°

Each execution displays

- Selected angle mode
- Input angle
- FPGA-computed SIN
- FPGA-computed COS
- FPGA-computed TAN
- Expected mathematical values
- Direct numerical comparison

This output verifies correct FPGA computation over multiple quadrants using degree inputs.

---

## Radian Mode

Image

```
output2
```

Demonstrates FPGA CORDIC execution using radian-based angle inputs.

Example test cases include

- 0.2617 rad
- 33.161° (internally converted to approximately 1.745 rad)

Each execution displays

- Selected angle mode
- Input angle in radians
- FPGA-computed SIN
- FPGA-computed COS
- FPGA-computed TAN
- Expected mathematical values
- Numerical comparison between hardware and software results

These examples verify that the firmware correctly converts, transmits, and retrieves radian-based CORDIC computations from the FPGA.

---

# Purpose

The screenshots contained in this directory demonstrate

- Successful RP2040 ↔ FPGA SPI communication
- Standalone firmware verification
- Degree-mode CORDIC computation
- Radian-mode CORDIC computation
- Comparison of FPGA results with software reference values
- Numerical validation of SIN, COS, and TAN before GUI integration
